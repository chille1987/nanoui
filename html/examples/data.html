Is NanoUI free to use?

Yes, NanoUI is MIT licensed and completely free for personal and commercial use. Install the gem, run the generator, and the code is yours.

Does it require a build step?

No. NanoUI uses vanilla CSS with native nesting and Stimulus controllers. No Tailwind, no PostCSS, no webpack. It works with Rails asset pipeline out of the box.

How do I customize the theme?

Edit the CSS custom properties in base/03_colors.css. All colors use HSL values, so changing your brand color is a single line edit. Dark mode is handled automatically.

Single-open
What browsers are supported?

NanoUI targets modern browsers: Chrome 120+, Firefox 117+, Safari 17.2+. These all support native CSS nesting and the <dialog> element.

Can I use it without Rails?

Absolutely. The CSS and Stimulus controllers are framework-agnostic. The Rails generators are optional convenience — you can copy the files manually into any project.

How does dark mode work?

Add the .dark class to your <html> element. All color tokens swap automatically via CSS custom properties. No JavaScript required for the theme itself.
