Is NanoUI free to use?

Yes! NanoUI is MIT licensed and completely free for personal and commercial use. Install the gem, run the generator, and the code is yours to keep and modify.

Does it require a build step?

No. NanoUI uses vanilla CSS with native nesting and Stimulus controllers. No Tailwind, no PostCSS, no webpack or Vite. It works with the Rails asset pipeline out of the box.

How do I customize the theme?

Edit the CSS custom properties in base/03_colors.css. All colors use HSL values, so changing your brand color is a single line edit. Dark mode updates automatically.

What browsers are supported?

Chrome 120+, Firefox 117+, Safari 17.2+. These all support native CSS nesting and the <dialog> element.
