Small Dialog

Max-width 24rem for compact content.

This is a small dialog ideal for confirmations or brief messages.

Default Dialog

Standard 32rem max-width for most use cases.

The default dialog size works well for displaying information, simple forms, and confirmation messages. The native <dialog> element provides built-in focus trapping and Escape key handling.

Large Dialog

Max-width 42rem for content-heavy dialogs.

Use the large variant when you need more space for complex forms, data tables, or detailed information. The dialog scales down on smaller viewports thanks to the calc(100% - 2rem) width constraint.
